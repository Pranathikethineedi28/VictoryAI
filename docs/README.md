# VictoryAI

VictoryAI is an autonomous AI career agent that discovers newly posted jobs, evaluates ATS compatibility, tailors resumes and cover letters, generates recruiter outreach messages, and prepares high-quality application packets.

## Features

- Job discovery across multiple career platforms
- ATS compatibility scoring
- Resume tailoring
- Cover letter generation
- LinkedIn message generation
- Cold email generation
- Application tracking
- Human approval workflow
- Future browser automation
